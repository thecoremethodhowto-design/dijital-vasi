# ⚠️ Sorumluluk Reddi Beyanı

## Genel Uyarı

Bu depoda yer alan yazılım ve belgeler **yalnızca eğitim ve araştırma amaçlıdır.**

## Üçüncü Taraf Yazılımlar

Bu proje aşağıdaki açık kaynaklı üçüncü taraf yazılımları kullanmaktadır:

- **OpenClaw** — Açık kaynak yerel ajan çerçevesi
- **Ollama** — Yerel model çalıştırma motoru
- **python-telegram-bot** — Telegram API istemcisi

Bu yazılımlar bağımsız projelerdir. Güncellemeler, güvenlik açıkları veya davranış değişiklikleri üzerinde kontrolümüz yoktur.

## Kullanıcı Sorumluluğu

Bu yazılımı kullanan kişi aşağıdakileri kabul etmiş sayılır:

- Sistemi **ana bilgisayarında değil**, izole bir ortamda (Docker konteyner veya sanal makine) çalıştırmak kullanıcının sorumluluğundadır.
- Telegram token ve ID bilgilerinin güvenliği **tamamen kullanıcıya aittir.**
- `.env` dosyasının GitHub veya başka bir platforma yüklenmemesi kullanıcının sorumluluğundadır.
- Asistanın ürettiği kod, rapor veya içeriklerin doğruluğu garanti edilmez. Kritik kararlar için üretilen çıktılar doğrulanmalıdır.
- Olası **veri kaybı, güvenlik ihlali veya sistem hasarından** The Core Method, kanal sahibi veya katkıda bulunanlar sorumlu tutulamaz.

## Yapay Zeka Çıktıları Hakkında

Bu sistem yerel yapay zeka modelleri kullanmaktadır. Modeller:

- Hatalı veya yanıltıcı bilgi üretebilir (halüsinasyon).
- Güncel bilgiye sahip olmayabilir.
- Bağlamı yanlış yorumlayabilir.

Üretilen hiçbir içerik profesyonel hukuki, tıbbi veya finansal tavsiye niteliği taşımaz.
